# Демо-сценарий для Todo Scanner MCP Server

## Цель сценария
Продемонстрировать работу двух основных инструментов MCP-сервера:
1. **`find_todos`** - поиск всех TODO/FIXME/BUG комментариев в проекте
2. **`analyze_tech_debt`** - анализ и приоритизация найденного технического долга

На тестовом проекте `demo_project/` мы увидим, как сервер находит несколько различных комментариев и правильно расставляет приоритеты (HIGH для критичных проблем, MEDIUM для обычных задач, LOW для незначительных пометок).

## Предусловия

### 1. Запус контейнера:
Выполните действия описанные в файле README.md и не переходите к пункту 2 пока не удостоверитесь что сервер работает

### 2. Запуск скрипта:
выполните программу "python test_demo.py" в новом терминале
