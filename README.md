# MCP Сервер: Todo-Scanner

## Что делает MCP-сервер
Этот сервер помогает разработчикам находить и приоритизировать технический долг в кодовой базе. Он сканирует файлы проекта и собирает все комментарии `TODO`, `FIXME`, `BUG` группируя их по файлам и типу.

**Какую боль решает:** Избавляет от необходимости вручную искать по проекту забытые пометки, помогая вести учет технического долга.

**Для кого полезен:** Для тимлидов, разработчиков, технических писателей, которые хотят иметь актуальный список задач по рефакторингу.

**Какие 2+ инструмента реализованы:**
1.  **`scan_for_todos`**: Сканирует указанную директорию и возвращает структурированный список всех найденных TODO/FIXME комментариев.
2.  **`prioritize_todos`**: (Пример второго тула) Анализирует найденные TODO и присваивает им приоритет (High/Medium/Low) на основе ключевых слов (например, "BUG" -> High, "optimize" -> Medium).

### Как собрать образ
docker build -t todo-scanner-mcp .

### Быстрый старт
docker run -p 8000:8000 todo-scanner-mcp

### Как проверить работу сайта
curl http://localhost:8000/health
В JSON должно быть: "status":"healthy"
